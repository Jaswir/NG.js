Aw yiss, the Newgrounds HTML5 API!
Get this code on Github. Created by NutcaseNightmare.
How to add this to your HTML5 game
  1. Create a new Newgrounds Game Project
  2. Download NG.min.js, the minified Javascript library.
  3. Put this at the end of your index.html page:
    <script src="lib/NG.min.js"></script>
  4. In your main Javascript file, call NG.connect like this:
    NG.connect(YOUR_API_ID,YOUR_ENCRYPTION_KEY);
  5. Shake your cursor like it's dancing, coz you're done! :D
How to Unlock Medals
Whenever you want a medal to be unlocked, call this in Javascript:
NG.unlockMedal(MEDAL_NAME);
It comes with its own UI, and will display even if your player is not logged in!
